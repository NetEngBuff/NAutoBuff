Dashboard Setup Required

Grafana is running but no dashboard URL is configured yet. Follow the steps below to create your network dashboard.

1

Open Grafana

Log in with admin / admin

Open Grafana → {{ grafana_base_url }}
2

Verify Datasource

Go to Connections → Data sources. You should see NAutoHUB-InfluxDB already configured (created by pilot.py). If missing, add an InfluxDB v2 datasource pointing to http://localhost:8086, org NAutoHUB, bucket NAutoHUB.

3

Create a Dashboard

Go to Dashboards → New → New dashboard. Add panels using Flux queries against the NAutoHUB-InfluxDB datasource. Example query: 

from(bucket: "NAutoHUB")
  |> range(start: -1h)
  |> filter(fn: (r) => r._measurement == "interface_counters")
4

Pin the Dashboard Here

Save the dashboard and copy its URL from your browser. Then stop NAutoHUB (Ctrl+C) and restart with the URL set: 

GRAFANA_DASHBOARD_URL="http://localhost:3000/d/your-id/name" ./pilot-config/run_nahub.sh
